How to use Encipherr ? :

  • To encrypt data, Enter your AES key in the key box. Then click Encrypt. To decrypt data, The same process as the encryption.

  • For file encryption, Enter your AES key in the key box, Select your file, Then click Upload and Encrypt, The same process for the decryption.

  • This means that if you want to read your own encrypted data, you'll need to have the same key that was used in the encryption.
